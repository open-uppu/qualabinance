# 💰 Finance Theory

> **The math meets the market. From instruments to derivatives, from portfolios to risk.**

## 🎯 What you need to know

### 📊 Portfolio Theory
- **Mean-Variance Optimization** (Markowitz 1952)
- **CAPM** (Sharpe 1964, Lintner 1965, Mossin 1966)
- **Multi-factor models** (Fama-French 1993, Carhart 1997, AQR)
- **Black-Litterman** (1992)
- **Risk Parity / ERC**
- **Hierarchical Risk Parity** (López de Prado 2016)

### 🎯 Derivatives Pricing
- **Black-Scholes-Merton** (1973)
- **Binomial tree** (Cox-Ross-Rubinstein 1979)
- **Monte Carlo** (Glasserman)
- **Stochastic volatility** (Heston 1993, Hull-White 1987)
- **Jump diffusion** (Merton 1976)
- **Interest rate models** (Vasicek 1977, CIR 1985)

### 📈 Volatility Modeling
- **GARCH family** (Bollerslev 1986, Nelson E-GARCH 1991, GJR 1993)
- **Realized volatility** (Andersen-Bollerslev 1998)
- **Stochastic volatility** (Heston 1993)
- **Implied volatility surface** (Gatheral)

### ⚠️ Risk Management
- **VaR / Expected Shortfall** (Artzner 1999, Rockafellar-Uryasev 2000)
- **Stress testing**
- **Drawdown analysis**
- **Deflated Sharpe Ratio** (Bailey-López de Prado 2014)
- **Purged CV** (López de Prado 2018)
