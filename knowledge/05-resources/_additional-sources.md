## 🗺️ How to use this catalog

1. **Build a daily reading routine** — 1 newsletter (15 min) + 1 paper (30 min) + 1 podcast episode (45 min)
2. **Build a weekly practice routine** — 1 Kaggle/Numerai submission + 1 GitHub repo exploration + 1 community post
3. **Build a monthly review** — re-check arXiv q-fin.ST/TR/GN/PR/RM + re-rank this catalog by what actually helped you
4. **Build a quarterly reset** — re-read AQR Insights + Robeco + Man Institute for the "what changed" view
